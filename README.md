## 快速开始

```bash
npm install
npm run build
npm run dev
```

默认 CLI 命令：

```bash
slime
slime chat
slime config
slime dashboard
```

## 配置

复制 `.env.example` 为 `.env`，按需填写你的模型与渠道配置。

本地用户配置文件会写入：

```bash
~/.slime/config.json
```
